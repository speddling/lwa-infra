# Home Lab

Personal Kubernetes home lab documentation, configurations, and manifests.

## Hardware
- **Control Plane / Workstation** - 
	- MacBook Air M4 running macOS (apex)
- **Worker Node** - 
	- AMD A10, ASRock AB350 Pro4, 16GB RAM, Fractal Design Define R4 (monolith)
- Monitoring Node - 
	- Asus VM40B, 8GB RAM, 1TB SSD (watchtower)

## Network
- TP-Link Omada ecosystem: 
- OC200 controller
- ER605 router
- TL-SG1210P PoE switch 
- 2x EAP245 APs

## Stack
- Kubernetes via k3s
- Terraform
- GitHub Actions
- Ansible
- Ubuntu Server 24.04 LTS on worker node
